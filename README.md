# contact-form

A simple contact form with validation

## Follow these steps to display the page

- Clone this repo
  ```bash
  git clone https://github.com/biniyamNegasa/contact-form.git
  ```
- Go into folder

  ```bash
    cd contact-form
  ```

- run
  ```bash
  npm install && npm run dev
  ```
